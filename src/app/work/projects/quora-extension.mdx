## Overview

The **Quora AI Extension** helps users generate high-quality answers with embedded promotions. It combines **GPT-based language models** with web automation.

## Features

- **Contextual Answer Generation** – uses GPT to create relevant, high-quality answers.  
- **Link Embedding** – seamlessly inserts promotional or affiliate links.  
- **Web + Dashboard Services** – manage generated answers and analytics.  
- **Customizable Prompts** – tweak tone, style, and CTA for each use case.  

## Impact

- Boosted productivity for marketers, who could generate answers 3–5× faster.  
- Enabled data-driven link placement for higher engagement.  
- Provided a scalable extension framework with dashboard monitoring.
